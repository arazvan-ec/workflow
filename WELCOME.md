## Tu Primer Comando

Tienes dos opciones para empezar:

### Opción A: Onboarding Guiado (Recomendado para nuevos usuarios)

```bash
/workflows:onboarding
```

Te guiaré paso a paso: verificación de prerequisitos, conceptos clave, y tu primera feature.

### Opción B: Empezar Directamente

```bash
/workflows:plan nombre-de-tu-feature
```

Ideal si ya conoces el workflow o quieres explorar por tu cuenta.

## Qué Incluye Este Plugin

```
21 agentes especializados  →  Cada uno experto en su área
25 comandos de workflow    →  Flujo completo de desarrollo
12 skills reutilizables    →  Automatización inteligente
```

### Novedades v2.5.0

```
Scoped Rules          →  Reglas que se cargan solo cuando son relevantes
CLAUDE.md slim        →  130 líneas (antes 514), contexto más eficiente
Routing Reference     →  Templates y decision matrix bajo demanda
Urgency calibration   →  Lenguaje claro sin fatiga de urgencia
```

### El Flujo en 4 Fases

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  PLAN   │───▶│  WORK   │───▶│ REVIEW  │───▶│COMPOUND │
│  80%    │    │  15%    │    │   4%    │    │   1%    │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
```

**Regla de oro**: 80% planificación, 20% ejecución.
